## Análise Comparativa das Atividades

### Questão 1: Loops (Java, Python, Haskell, Swift)
* **Quantidade de linhas: Python e o Swift foram os mais curtos. O Java é um pouco mais longo por causa da burocracia dos tipos, e o Haskell ficou maior por causa da função extra.** 
* **Legibilidade e Concisão: O Java, para mim, é o mais legível porque já trabalho com ele, mas o Python é muito mais direto.** 
* **Opinião Pessoal: Usei o Java por ser minha linguagem do dia a dia, então foi a mais fácil de escrever e ler. O Haskell foi o pior, pois transformar um loop simples em recursão foi um grande problema.** 
* **Comparação: Comparado ao C, o Swift parece uma versão muito mais moderna e amigável.** 

### Questão 2: Seleção Múltipla (C, Ruby, Erlang)
* **Quantidade de linhas: Com Ruby resolvi tudo com pouquíssimas linhas. O C ficou enorme por causa dos vários cases e breaks.** 
* **Legibilidade: O Ruby parece um texto em inglês, muito fácil. O Erlang foi o mais estranho de ler por causa da pontuação.** 
* **Opinião Pessoal: O Ruby é muito mais limpo. O case dele deixa o código organizado, enquanto o switch do C fica todo poluído com aquele monte de break** 
* **Comparação: Comparando com o Java que eu uso, o Ruby é muito mais prático para agrupar valores.** 

### Questão 3: Sem Goto (Java, Go, PHP)
* **Quantidade de linhas: As três ficaram bem parecidas, já que tive que trocar o switch por if/else em todas.** 
* **Legibilidade: O Go pareceu o mais limpo visualmente por não ter tantos parênteses, mas o Java continua sendo melhor para entender a lógica.** 
* **Opinião Pessoal: Adaptar a lógica sem break em Java, Go e PHP foi trabalhoso. Achei que o código ficou pior e mais difícil de ler, pois a condição do for ficou muito "suja".** 

### Questão 4: For Multilinguagens (PHP, Go, Swift, Lua, C++)
* **Quantidade de linhas: C++ e PHP foram idênticos ao Java original. Swift e Lua deram mais trabalho porque tive que controlar uma das variáveis na mão dentro do loop.** 
* **Legibilidade:  O C++ e o PHP ganham porque você vê tudo o que o loop faz logo no cabeçalho.**
* **Opinião Pessoal: A linguagem que achei mais estranha foi o PHP, pois nunca tinha tido contato com ela antes. A sintaxe é muito diferente das outras linguagens que eu já vi.** 
* **Comparação: Achei curioso como o Go resolve as duas variáveis no for de um jeito diferente.**
